Examen Git y Github

Alumn@: Daniel Héctor Barbieri

  1. Que es un sistema de control de versiones?

    Es un sistema donde los cambios que se realizan en los distintos archivos del proyecto quedan guardados, para poder acceder a cualquier punto del mismo y recuperar la información de ese momento.

  2. Cual es la diferencia entre git y github?

    Git es un controla de versiones, en cambio github es un lugar en donde podemos alojar nuestros proyectos.

  3. Cómo creo una rama en git?

    Para crear una rama en git hay que ir a la rama de la cual queremos partir con el comando: git checkout [nombre_rama] Ahora debemos ejecutar el comando git checkout -b [nombre_rama_nueva]

  4. Cómo clono un repositorio?

    Para clonar un repositorio se debe ejecutar el comando: git clone [url_repo]
