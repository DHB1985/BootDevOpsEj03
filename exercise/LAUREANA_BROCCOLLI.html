Examen Git y Github

Alumn@: Laureana Broccolli

  1. Que es un sistema de control de versiones?

    Un sistema de control de versiones es un sistema el cual registra los cambios realizados en uno o mas archivos. De modo que se pueda recuperar versiones especificas en un futuro.

  2. Cual es la diferencia entre git y github?

    Git nos ayuda a controlar las versiones de cada archivo, en cambio Github es la plataforma en la cual podemos subir los archivos.

  3. Cómo creo una rama en git?

    Para crear una nueva rama tenemos que primero posicionarnos en la carpeta donde queremos crear la nueva rama, (cd + nombre de la carperta) luego colocamos git checkout -b + e nombre de la rama, y ya tenemos creada nuestra nueva rama.

  4. Cómo clono un repositorio?

    Para clonar un repositorio, vamos al repo que queremos clonar y copiamos el url. Luego colocamos git clone + el url del repo y listo!
